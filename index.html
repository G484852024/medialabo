各授業会回の学習事項

第1回 WWW と HTML

URLはプロトコル、サーバ、パスの3つの部分で構成されている。Webページを構成する部品は大きく「HTML」「CSS」「JavaScript」「画像・音声・動画」の4つに分けられる
HTMLは要素を組み合わせて作ることができる。要素はブロック要素とに分けられる。インライン要素に分けられる。 また、要素に追加する情報を属性という。
HTMLを作成する際は「<!DOCTYPE html>」「<html>」と毎回書く必要がある。
演習の成果:演習1-4の成果

第2回 CSS による Web ページの装飾

htmlの見た目を変える記述をcssという。htmlのhead要素に外部スタイルシートのファイルを指定する。
htmlの要素を指定し、{}内に指示を与える。
枠の中に文を記入する際は中に入る文の要素にclass属性を付けその属性をcssで指定する。
演習の成果: 演習2-3の成果

第3回 JavaScript の基本

ハイパーリンク（通称「リンク」）によって Web ページが接続されていて，ユーザはリンクを辿ってたくさんの Web ページを見ることができる．
webページを構成する部品は大きく4つに分けられる。「HTML」「CSS」「JavaScript」「画像・音声・動画」
HTMLはwebページの構造を指定するための記述でページは見出しや本文,箇条書き,図表などでできている,このようなページの構造を記述するのがHTMLだ。
演習の成果: 演習3-6の成果

第4回 DOM と JavaScript

Html文書内にInput要素を作ることで入力項目を作ることができる。Input要素のtype属性は入力する内容によって変える。
Size属性は入力するテキストボックスのサイズを変更する。Input要素の前にlabel要素を置き入力項目を説明する。Button要素を置くことで画面にボタンを設置する。
document.querySelector(ここに要素を入れる)でhtml内の探したい要素を検索できる。
しかし、複数ある場合最初のものしか選べないdocument.querySelectorAll(ここに要素名)で全て選ぶ。
演習の成果: 演習4-10の成果

第5回 イベント処理

javaScriptを使うときは絶対html文書の指定したjavaScriptのページの後に空白を開けdeferと書く。これ書かないと動かない。まず確認する。
javaScriptでhtmlを変えたいときは要素を作る{createElement(要素名)}と要素の中に要素を配置する{要素.insertAdjacentElement(‘beforeend’,中に配置したい要素)}と
要素の後に要素を配置する{要素.insertAdjacentElement(‘afterend’,後におきたい要素)}と要素に属性をつける{要素.setAttribute(つけたい属性,属性の値)}と
要素を消す{消したい要素.remove()}を組み合わせて行う
演習の成果:演習5-7の成果
